# Pluginization_Of_Flutter
Flutter 插件化开发架构，内部有使用Agent Skills 配置，本项目结合个人flutter开发经验以及Claude Code 开发组件

## 目录结构

### DOCS
  项目文档文件夹

### flutter_earth_online
  项目源码文件夹
####  apps
    项目主工程文件夹
  ####  lib
    项目入口
  ####  packages
    项目依赖插件文件夹
  #####  bridge
    桥接层文件夹
  #####  core
    核心项目基础配置层文件夹
  ##### features
    项目功能模块文件夹
  ##### im_kit
    即时通讯插件文件夹(可以通过AI工具接入完整SDK 或者 开源服务)

  #### infrastructure
    基础设施层文件夹
  #### router
    路由层文件夹
  #### ui_kit
    UI组件层文件夹
  #### shared
    公共组件层文件夹
